Selected Projects

My work spans both the study of theoretical concepts as well as applications of those concepts, which led to projects as diverse as system design, machine learning, data quality analysis, data storage and data indexing, or location tracking.

Mobility Collector: a tracking platform

The Mobility Collector is an open-source platform designed specifically for collecting high quality mobility specific data in a battery conscious manner.
The project was presented at the Mobile Ghent 2013 conference.

Comfort Zones Trajectory Mining

Comfort Zones Trajectory Mining is a database system that supports automatic classification and clustering of trajectories using the postGIS extension of postgres. Developed as part of a project assignment for the course Modern Database Systems and Their Applications (KTH).

Location Drops

Location Based Android Application: A location based service for arranging meetings between users combining social networking and navigation. Developed as part of a project assignment for the course Developing Mobile Applications (KTH).

Open Science Map

OpenScienceMap provides free and open maps for Android with the fastest and 100% pure vector maps around. It is a platform used to enable researchers to implement their ideas, to cooperate with others, and to share their results with everybody.
